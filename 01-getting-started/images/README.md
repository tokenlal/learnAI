# Chota Bheem Character Images

## Required Images for the Tutorial

Add the following PNG images to this directory:

1. **bheem.png** - Main character, strong and brave
2. **chutki.png** - Smart and resourceful friend
3. **raju.png** - Clever and strategic friend
4. **kalia.png** - Competitive rival who becomes friend
5. **dholu.png** - One of the twins
6. **bholu.png** - Other twin
7. **jaggu.png** - Bheem's monkey friend

## Recommended Specifications

- **Format:** PNG with transparent background
- **Size:** 400x400 pixels (square)
- **Resolution:** 72-150 DPI
- **File size:** < 200KB each for fast loading

## Where to Get Images

### Option 1: Official Licensing (Recommended for Commercial Use)
Contact Green Gold Animation:
- Website: https://www.greengold.tv
- Request licensing for educational/commercial use
- They provide high-quality official artwork

### Option 2: Free Resources
Search for:
- "Chota Bheem PNG transparent" on Google Images
- Check licensing - look for Creative Commons or educational use
- Fan art communities (with proper attribution)

### Option 3: Screenshot and Edit
- Take screenshots from episodes (for personal/educational use only)
- Use tools like remove.bg to remove backgrounds
- Crop to square format

### Option 4: Create Simple Graphics
- Use Canva or similar tools
- Create simple character avatars inspired by the show
- Use characteristic colors: Orange for Bheem, Pink for Chutki, Blue for Raju

## Quick Setup

Once you have the images:

```bash
# Place all 7 PNG files in this directory
# The HTML will automatically load them

ls images/
# Should show:
# bheem.png  chutki.png  raju.png  kalia.png  dholu.png  bholu.png  jaggu.png
```

## Testing

Open `index.html` in your browser - if images are present, they'll display in the character showcase. If missing, emoji placeholders will show instead.
