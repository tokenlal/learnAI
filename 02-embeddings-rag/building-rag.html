Key Components Overview üîß

üß†

embedding_utils.py

Purpose: Local embedding generation using Sentence Transformers

  • Loads all-mpnet-base-v2 model locally
  • Generates 768-dimensional embeddings
  • No API calls or external dependencies
  • Consistent, cost-free processing
‚úÇÔ∏è

chunk_utils.py

Purpose: Intelligent document chunking and preprocessing

  • Splits documents into optimal-sized chunks
  • Preserves context with overlap handling
  • Maintains document metadata and relationships
  • Handles various text formats efficiently
üóÑÔ∏è

milvus_client.py

Purpose: Vector database operations and management

  • Creates and manages Milvus collections
  • Handles vector insertion and indexing
  • Performs similarity search with L2 distance
  • Manages schemas and metadata filtering
üöÄ

main.py (FastAPI)

Purpose: RESTful API endpoints for RAG operations

  • /upload: Process and store documents
  • /query: Search and generate answers
  • /health: System status monitoring
  • Auto-generated API documentation
üèóÔ∏è How Components Work Together
  1. Document Upload: Text is processed through chunk_utils.py
  2. Embedding Generation: embedding_utils.py creates vectors locally
  3. Storage: milvus_client.py stores vectors with metadata
  4. Query Processing: FastAPI coordinates search and generation
  5. Response: Formatted JSON with answers and source citations

Getting Started with Our RAG System üöÄ

1 Navigate to RAG Project

Command:
cd /path/to/learnAI/rag_poc

The complete, production-ready RAG system is located in the rag_poc/ folder. This contains all the components you need to run a scalable document Q&A system.

2 Start the System

Docker Deployment:
./run-with-rancher.sh up

Manual Setup:
pip install -r requirements.txt
python app/main.py

The system will start:

  • Milvus: Vector database on port 19530
  • FastAPI: REST API on port 8000
  • Auto-docs: Available at http://localhost:8000/docs

3 Test the API

Upload Document:
POST /upload with text file

Ask Questions:
POST /query with {"query": "Your question here"}

Check Health:
GET /health

The system processes documents automatically and provides answers with source citations.
