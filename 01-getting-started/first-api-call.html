üéØ The Simple 4-Step Process
  1. Import: Load the OpenAI tools and your API key
  2. Initialize: Create a connection to OpenAI
  3. Send: Make your API call with your question
  4. Receive: Get and display the AI's response

Your First API Shot! üèπ

1 Create first_call.py:

import os
from dotenv import load_dotenv
from openai import OpenAI

# Load environment variables
load_dotenv()

# Initialize OpenAI client (Raju preparing his bow!)
client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

# Make your first API call (Shooting the arrow!)
response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "user", "content": "Say hello and introduce yourself in one sentence!"}
    ]
)

# Print the response (Target hit!)
print("üéØ AI Response:")
print(response.choices[0].message.content)

2 Run your first AI program:

python first_call.py

üéØ Parameters Explained
  • model: Which AI brain to use (gpt-3.5-turbo is fast & cheap!)
  • messages: Your conversation messages
  • role: Can be "user" (you), "assistant" (AI), or "system" (instructions)
  • content: Your actual message text

Adding Instructions üé≠

Guide the AI's behavior with system messages:

response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {
            "role": "system", 
            "content": "You are Raju - clever, strategic, and helpful!"
        },
        {
            "role": "user", 
            "content": "Give me a quick coding tip!"
        }
    ]
)

print(response.choices[0].message.content)
üé≠ Message Types
  • system: Sets AI's personality and style
  • user: Your actual question
  • assistant: Previous AI responses (for conversation memory)

Controlling Output üéõÔ∏è

Fine-tune how AI responds:

response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "user", "content": "Write a short poem about Dholakpur"}
    ],
    temperature=0.8,      # Creativity (0-2, default 1)
    max_tokens=100,       # Maximum response length
)
üéØ Key Parameters
  • temperature: Controls creativity
    • 0 = Very focused, same every time
    • 1 = Balanced (default)
    • 2 = Super creative, different each time
  • max_tokens: Limits response length (1 token ‚âà 0.75 words)

import os
from dotenv import load_dotenv
from openai import OpenAI

load_dotenv()
client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

try:
    response = client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=[
            {"role": "user", "content": "Hello from Dholakpur!"}
        ]
    )
    print("‚úÖ Success!")
    print(response.choices[0].message.content)
    
except Exception as e:
    print(f"‚ùå Error: {e}")
    print("üí° Raju suggests: Check your API key!")
‚ö†Ô∏è Common Problems
  • AuthenticationError: Wrong API key
  • RateLimitError: Too many requests (wait a bit!)
  • APIError: OpenAI service issue

Tracking Costs üí∞

See how much each request costs:

response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "user", "content": "Hello!"}
    ]
)

# Check token usage
usage = response.usage
print(f"üìä Tokens:")
print(f"   Question: {usage.prompt_tokens}")
print(f"   Answer: {usage.completion_tokens}")
print(f"   Total: {usage.total_tokens}")

# Calculate cost
cost = (usage.total_tokens / 1000) * 0.002
print(f"üí∞ Cost: ${cost:.6f}")
