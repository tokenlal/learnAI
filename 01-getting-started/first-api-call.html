Understanding the AI Connection üîå

Snape observes: "Even a first-year should comprehend this. It's merely sending instructions and awaiting a response. Do try not to complicate it unnecessarily."

The OpenAI API is like sending an owl to a very smart wizard. We ask a question, it thinks, and sends back a helpful answer. Let's learn how to do this properly!

ü™Ñ The Simple 4-Step Process
  1. Import: Load the OpenAI tools and your API key
  2. Initialize: Create a connection to OpenAI
  3. Send: Make your API call with your question
  4. Receive: Get and display the AI's response

Your First Magical API Call! ü™Ñ

1 Create first_call.py:

import os
from dotenv import load_dotenv
from openai import OpenAI

# Load environment variables
load_dotenv()

# Initialize OpenAI client (Ron preparing his wand!)
client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

# Make your first API call (Casting the spell!)
response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "user", "content": "Say hello and introduce yourself in one sentence!"}
    ]
)

# Print the response (Spell succeeded!)
print("ü™Ñ AI Response:")
print(response.choices[0].message.content)

2 Run your first AI program:

import os
from dotenv import load_dotenv
import google.generativeai as genai

load_dotenv()
genai.configure(api_key=os.getenv("GEMINI_API_KEY"))

try:
    # Initialize model with settings
    model = genai.GenerativeModel(
        'gemini-2.0-flash',
        generation_config={
            "temperature": 0.7,      # Creativity control (0-2)
            "max_output_tokens": 150,  # Response length limit
        }
    )
    
    # Generate content
    response = model.generate_content(
        "Write a short motivational quote about learning AI"
    )
    
    print("‚úÖ Success!")
    print(response.text)
    
except Exception as e:
    print(f"‚ùå Error: {e}")

Understanding the Code üìñ

Part 1: Setting Up

from openai import OpenAI
client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

Ron explains: "This creates our connection to AI, like having your wand at the ready!"

Part 2: The API Call

response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "user", "content": "Your question here"}
    ]
)
ü™Ñ Parameters Explained
  • model: Which AI brain to use (gpt-3.5-turbo is fast & cheap!)
  • messages: Your conversation messages
  • role: Can be "user" (you), "assistant" (AI), or "system" (instructions)
  • content: Your actual message text

Part 3: Getting the Response

print(response.choices[0].message.content)

Ron explains: "This extracts the AI's answer, like reading the reply from your owl!"

Adding Instructions üé≠

Guide the AI's behavior with system messages:

response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {
            "role": "system", 
            "content": "You are Ron - loyal, practical, and helpful!"
        },
        {
            "role": "user", 
            "content": "Give me a quick coding tip!"
        }
    ]
)

print(response.choices[0].message.content)
üé≠ Message Types
  • system: Sets AI's personality and style
  • user: Your actual question
  • assistant: Previous AI responses (for conversation memory)

Controlling Output üéõÔ∏è

Fine-tune how AI responds:

response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "user", "content": "Write a short poem about Hogwarts"}
    ],
    temperature=0.8,      # Creativity (0-2, default 1)
    max_tokens=100,       # Maximum response length
)
ü™Ñ Key Parameters
  • temperature: Controls creativity
    • 0 = Very focused, same every time
    • 1 = Balanced (default)
    • 2 = Super creative, different each time
  • max_tokens: Limits response length (1 token ‚âà 0.75 words)

Tracking Costs üí∞

See how much each request costs:

response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "user", "content": "Hello!"}
    ]
)

# Check token usage
usage = response.usage
print(f"üìä Tokens:")
print(f"   Question: {usage.prompt_tokens}")
print(f"   Answer: {usage.completion_tokens}")
print(f"   Total: {usage.total_tokens}")

# Calculate cost
cost = (usage.total_tokens / 1000) * 0.002
print(f"üí∞ Cost: ${cost:.6f}")
